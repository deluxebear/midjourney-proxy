# midjourney-proxy
全功能Midjourney Discord代理，支持Midjourney所有功能，高性能，稳定，免费。

如果喜欢，请给个Star⭐️，谢谢！

## 支持功能

- [x] 支持 Imagine(画图)
- [x] 支持 Imagine 时支持添加图片垫图
- [x] 支持扩图 Pan ⬅️ ➡️ ⬆️ ⬇️
- [x] 支持扩图 ZoomOut 🔍
- [x] 支持自定义扩图 Custom Zoom 🔍
- [x] 支持局部重绘 Vary (Region) 🖌
- [x] 支持 Make Square
- [x] 支持任务实时进度
- [x] 支持 Blend(图片混合)
- [x] 支持 Describe(图生文)
- [x] 支持账号池
- [x] 支持禁用词设置
- [x] 支持图片cdn替换
